# EthiScan — AI Powered Ethical Brand Intelligence Platform

## 1. Project Overview

EthiScan is an AI-powered ethical intelligence platform that analyzes brands in real time using live internet data. Instead of relying on static datasets, EthiScan gathers sustainability, labor, environmental, and ethical signals from public web sources and generates dynamic ethical evaluations using AI.

The platform helps users make more informed purchasing decisions by providing:

* Ethical scores
* Sustainability insights
* Positive indicators
* Ethical concerns
* Smarter ethical alternatives

EthiScan combines live web intelligence, AI reasoning, and interactive analytics into a modern ethical transparency dashboard.

# 5. Installation

## Clone Repository

```bash
git clone https://github.com/your-username/EthiScan.git
```

## Navigate to Backend

```bash
cd EthiScan/backend
```

## Install Dependencies

```bash
npm install
```

## Start Server

```bash
node server.js
```

## Open Application

```text
http://localhost:5000
```

---

# 6. Environment Variables

Create a `.env` file inside the backend folder.

```env
OPENROUTER_API_KEY=your_openrouter_key
SERPER_API_KEY=your_serper_key
JWT_SECRET=your_secret_key
MONGO_URI=your_mongodb_connection_string
```
